# くきおmod

Agar.io向けのESign注入用MODです。

## 機能

- 日本語メニュー
- 起動時に「おつかれっす～」を約2秒間表示
- 起動メッセージ、情報表示、メニュー全体にRounded M+ 1c Boldフォントを使用
- メニューアイコンを背景透過済みのゾロダック画像に変更
- マスタースプリット設定ページ
- 分裂ループのON/OFF
- 分裂間隔を0.00秒から0.50秒で調整
- 追加分裂ボタンの表示/非表示
- ボタン編集モード
- 追加分裂ボタンのドラッグ移動
- 追加分裂ボタンサイズ調整
- 追加分裂ボタン位置リセット
- 追加分裂ボタンをAgar.io本来の分裂ボタン画像に変更
- 視野設定ページ
- 分裂視野
- 固定視野
- 画面上の+/-ボタンのタップまたは長押しでズーム倍率を0.01ずつ調整
- ズーム倍率を0.02倍から2.00倍で調整
- 長押し中は0.10秒ごとに倍率変更
- 視野ボタン編集モードONの時だけ+/-ボタンをドラッグ移動
- 視野+/-ボタンサイズ調整
- ラベル設定ページ
- 敵スコアON/OFF
- 視野内の敵セルと仲間セルに質量ラベルを表示
- 敵スコアラベルサイズを0.5倍から2.0倍で調整
- 大きさ確認ON/OFF
- 敵スコアの色で大きさ判定を表示
- 判定基準は自分の一番大きいセルの半径。半径が取れない場合だけ質量で判定
- 黄: 1回分裂で飛んだ後に重なれる大きさ
- 緑: 1回分裂で食べられる
- 青: 2回分裂後のセルでも食べられる
- 赤: 相手が1回分裂したら自分が食べられる
- カスタムスキン作成サイトを開く
- グリッチとしてXを開く
- お絵描き依頼としてXを開く
- メニューからAgar.io風フルスクリーン背景と落下位置点線、次ブロック表示付きのテトリスを起動
- メニューからドラッグで分裂方向を決め、遠くで互いに食べ合う敵連携の受け渡しの隙を4分裂/8分裂で狙う分裂練習を起動
- メニューからスキン画像の一部を見て名前を当てる4択スキンクイズを起動
- スキンクイズは現在の正解数表示、正解発表時の全体画像表示、難易度切替に対応
- ゲームプレイ中にメニューアイコンを1秒長押しすると、パーティーコードとゲームモードを表示
- パーティーコードは作成時保存だけでなく、現在のパーティー表示ラベルから自動検出
- 検出したパーティーコードを保存し、画面を開かず入り直した時も前回コードを表示

## 対象

- Bundle ID: `com.miniclip.agar.io`
- Agar.io 26.6.0の `ControlsWidget` / `splitButtonCallback` を利用
- 視野変更は `BaseArenaView` の `calculateZoom:cellAmount:` を利用
